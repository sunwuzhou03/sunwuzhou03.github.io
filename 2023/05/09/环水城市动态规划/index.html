环水城市最短路径条数

题目描述

伪代码

题目中所给的是从左下角到达右上角，我们在此可以转换成从右上角到达
左下角，并转变大厦的布局。转换后的问题和原问题等价。至此简化了问
的求解

输入： 城市大小mn,城市大厦右上角坐标x,y以及垂直方向上占用
小房子的数量z和水平方向上占用的小房子数量b.
*输出： 从左下角到达右下角最短路径的条数

  1. mp = {0,0,0…};//初始化地图mp数组全部元素为0
  2. dp = {0,0,0…};//初始化动态规划dp数组全部元素为0
  3. for i←x+1 to x+z-1
  4. for j←y+1 to y+b-1
